
# gpt4 cli

### usage

```sh
gpt4 "your prompt"
```

### installation

install with cargo
```sh
cargo install gpt4-cli
```

add api key to `$HOME/.config/gpt4-cli/config.toml`
```toml
openai_api_key="your api key"
```

full send!
